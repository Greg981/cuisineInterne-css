Syntaxe CSS
CSS est composé de règles de style que le navigateur interprète puis applique aux éléments correspondants de votre document.
Une règle de style comporte des éléments de syntaxe à respecter:

Voila à quoi ça doit ressembler:

p {
color: red;
font-size:130%;
}

Le point à retenir :

Une déclaration CSS se termine toujours par un point-virgule et les groupes de déclarations sont entourés d'accolades.
