Prenons l'exemple de ce texte : Comment vais je l'écrire????

Imaginons que nous voulons: le souligner, le mettre en gras, agrandir la police de caractère
et le mettre en rouge

naturellement tout ça en meme temps sinon ça serait pas marrant

Comment vais je l'écrire????

Plusieurs moyens s'offrent à nous :

  1. via HTML uniquement:

    2. La façon de proceder va etre de definir chaque éléments en code HTML :

    Ce qui donne:

    <u>pour le souligner</u> ; <b> pour le mettre en gras</b> ; <font size="+3"> pour agrandir la police de caractère</font> ; <font color="red">pour et le mettre en rouge</font>

    <font color="red"><font size="+3"><u><b>Comment vais je l'écrire????</b></u></font></font>

    Pas des plus convivial non ??? On se retrouve vite avec une ligne de code tres compliqué à gérer
    Et ce pour une seule ligne écrite....
    Voyons donc l'autre méthode

  2. Tout en utilisant le language HTML pour créer nos boites (div) et combinant le CSS pour gérer les attributs et styles de nos boites:

    3. La façon de proceder va etre la suivante :

    1. de créer en HTML une div class (une boite) que j'ai nommé: texthtml1
      Ce qui donne: (j'avoue j ais mis de la couleur sinon ca ressemble à rien)
      <div class=texthtml1>

      </div>

    2. d'écrire notre texte dans cette div
      Ce qui donne en HTML:
      <div class=texthtml1>

      <p> Comment vais je l'écrire????</p>

      </div>
    3. de créer le nom de cette div (texthtml1) dans CSS et lui donner les attributs requis
      Ce qui donne en CSS:

      .texthtml1 {

      font-family: Arial, Helvetica, sans-serif, cursive;
      font-size: 25px;
      text-decoration: underline;
      font-weight: bold;
      color: red;
      background-color:rgb(148, 132, 132);
      padding: 10px;
      margin: 5px;
      }
    4. et ensuite miracle nous voyons que les attributs entrée en CSS mettent en forme notre texte qui est contenu dans notre div.
      Ce qui donne comme comme résultat:

      Comment vais je l'écrire????

      Vous imaginez pas combien de div j'ais créé pour cette simple page . L'art et la manière de passer son week end à faire une présentation....

      Le point à retenir :

      Les feuilles de style sont destinées à séparer la structure de la page et sa mise en forme.
      Elles permettent des effets impressionnants impossibles à réaliser en HTML.
